# Anime Tosho Hayase Extension

Hayase extension manifest for Anime Tosho torrent and NZB support.

## Install

In Hayase:

1. Open `Settings -> Extensions -> Repositories`.
2. Paste this manifest URL:

```text
https://raw.githubusercontent.com/rurin1x/animetosho-hayase-extension/main/index.json
```

3. Click `Import Extensions`.

Do not paste the GitHub repository page URL. Hayase expects a direct manifest file URL.

## Sources

- `Anime Tosho` - torrent search via Anime Tosho JSON API.
- `Anime Tosho (Usenet)` - NZB lookup by torrent info hash.
